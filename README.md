##  KANBAR-BOARD 

A light weight javascript project that demonstrate how to drag and drop items from one place to another.

------

## Features

1. Drag cards between lists
2. Visual feedback when hovering over a drop zone
3. Cards move between list usinf HTML drag and drop
4. Simple javascript no any external library required

-------

##  Technology

- **HTML**
- **CSS**
- **JAVASCRIPT**
